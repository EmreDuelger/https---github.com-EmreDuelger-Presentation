Wieso gibt es das DOM?

  • World Wide Web gewinnt an Popularität und Javascript wird erfunden
  • Browserhersteller erfinden eigene Modelle, die die Änderung der Struktur und des Aussehens ermöglichen
  • Web-Programmierer müssen Webseiten für die zu unterstützenden Browser anpassen (Mehrfachprogrammierung)

Was ist das DOM?

Definition

  • World Wide Web Consortium Standard
  • plattform- und sprachenunabhängige Programmierschnittstelle
  • Dokumente bestehen aus Knoten (nodes) in einer hierarchischen Baumstruktur
  • dynamischer Zugriff auf Inhalt, Struktur und Style [1]

Beispiel: JavaScript und DOM

  • DOM stellt abstrakte Interfaces wie z.B. Document, Node und Element zur Verfügung
  • JavaScript muss diese Interfaces implementieren um auf das document-Objekt zugreifen zu können
  • Das bedeutet: JavaScript kann, wenn es die DOM-API implementiert hat, auf die Elemente des document-Objekts dynamisch zugreifen [2]

Beispielhafte DOM-Struktur [3]

wichtige Zugriffsmethoden auf Elemente[4]

  • document.getElementById()
  • document.getElementsByName()
  • document.getElementsByClassName()
  • document.getElementsByTagName()

Anwendungsbeispiel


function bildEinfügen(){
    var tabelle = document.getElementById("Tabelle");
    var überschrift = document.getElementById("Überschrift");
    var eigentlicheTabelle = document.getElementById("EigentlicheTabelle");
    var anderesElement = document.createElement("div");

    // Neues Element erstellen und stylen
    anderesElement.innerHTML = "<h2>DOM manipuliert!</h2>";

    // DOM-Operationen: Ersetzen und Entfernen
    überschrift.parentNode.replaceChild(anderesElement, überschrift);
    eigentlicheTabelle.parentNode.removeChild(eigentlicheTabelle);
    tabelle.parentNode.removeChild(tabelle);
}
