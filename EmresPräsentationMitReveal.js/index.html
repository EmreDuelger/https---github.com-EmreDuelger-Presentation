Wieso gibt es das DOM?

  • World Wide Web gewinnt an Popularität und Javascript wird erfunden
  • Browserhersteller erfinden eigene Modelle, die die Änderung der Struktur und des Aussehens ermöglichen
  • Web-Programmierer müssen Webseiten für die zu unterstützenden Browser anpassen (Mehrfachprogrammierung)

Was ist das DOM?

Definition

  • World Wide Web Consortium Standard
  • plattform- und sprachenunabhängige Programmierschnittstelle
  • Dokumente bestehen aus Knoten (nodes) in einer hierarchischen Baumstruktur
  • dynamischer Zugriff auf Inhalt, Struktur und Style [1]

HTML DOM definiert...

  • HTML Elemente als Objekte
  • Eigenschaften von HTML Elementen
  • Methoden um Zugriff auf HTML Elemente zu bekommen
  • Ereignisse für HTML Elemente

Beispiel: JavaScript und DOM

  • DOM stellt abstrakte Interfaces wie z.B. Document zur Verfügung
  • JavaScript muss dieses Interface implementieren um auf das document-Objekt zugreifen zu können
  • Das bedeutet: JavaScript erhält die Methoden des document-Objekts implementieren und bekommt dessen Eigenschaften [2]

Beispielhafte DOM-Struktur [3]

wichtige Zugriffsmethoden auf Elemente

  • document.getElementById()
  • document.getElementsByName()
  • document.getElementsByClassName()
  • document.getElementsByTagName()
